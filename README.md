# ModernTech Solutions - HR Management Portal

## Project Description
A purely front-end Human Resource web application built as a proof of concept for ModernTech Solutions. The portal centralizes employee record management, automates client-side payroll calculations, and tracks time-off requests using dummy data and local persistence.

## Features
- **Employee Data Management:** Profile creation and data tracking for a minimum of 15 records.
- **Automated Payroll:** Client-side calculations of monthly and annual salaries, generating digital payslips.
- **Attendance & Leave Tracker:** Client-side time-off submission, approval, and real-time attendance status updating.
- **System Preferences Dashboard:** POPIA 2021 compliant setting panels.

## Technologies Used
- Frontend Framework: Vue.js 3
- Styling: Tailwind CSS / Bootstrap 5
- Local Data Management: JavaScript localStorage
